# chess

## general

запуская проект шахматы, пользователь попадает в меню, где он может загрузить начатую игру из json-файла или начать новую. Игра проходит в соответствии с обычными правилами

## mate & stalemate

поставив мат или пат, пользователь видит messagebox с результатом игры. в этом случае при закрытии формы результат не будет сохранен в json.
